Qué hace la app:

Múltiples vacas en pestañas — puede agregar tantas como necesite y comparar parámetros lado a lado
Dos modos de entrada: DEL numérico directo, o fechas de control (calcula automáticamente los DEL desde la fecha de parto)
Pega desde Excel directamente en el área de texto — acepta tab, coma o punto y coma como separador, y coma decimal o punto
Ajuste del modelo: usa mínimos cuadrados linealizados (ln Y = ln a + b·ln t − c·t), exactamente el método del artículo de Cañas et al. (2009)
Resultados: a, b, c, t_pico, y_max, s (persistencia), R², RMSE, tabla de interpretación con rangos de referencia para Holstein y BON×Holstein en trópico alto
Gráfica con datos observados, curva ajustada y línea vertical en el pico
Exportar CSV con los parámetros y los residuos para cada observación
Carga de ejemplo con datos típicos de una Holstein para probar de inmediato
